# Price Formatter

The script formats any inputed price to useful view


# Quickstart

Run **format_price.py** with argument. 

```bash

$ python format_price.py 6844389.32

Formatted price:
6 844 389.32

```

or import format_price() function

```python 
from format_price import format_price
format_price(price)
```
